# CodeWhale Homebrew Tap

Homebrew formula for [CodeWhale](https://github.com/tanuki-cat/CodeWhale) —
a terminal-native coding agent for any model, open models first.

## Install

```bash
brew tap tanuki-cat/codewhale
brew install codewhale
```

## Update

```bash
brew upgrade codewhale
```

## Uninstall

```bash
brew uninstall codewhale
brew untap tanuki-cat/codewhale
```
